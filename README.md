# BoomBot

## Installation

### Requirements

1. node.js
2. npm
3. discord.js
4. Using opusscript or node-opus, ytdl-core
5. simple-youtube-api

## Usage

**All Operating Systems**

```
cd [project directory]
```
Then
```
node bot.js
```
To stop just type stop into the shell and enter.

## Versions

### v1.0
Commands include:
* np (now playing)
* pause
* ping
* play
* queue
* resume
* skip
* stop
* vol (volume)
