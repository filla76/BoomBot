# BoomBot

## Installation

### Requirements

1. node.js
2. npm
3. discord.js
4. Using opusscript or node-opus, ytdl-core
5. simple-youtube-api

## Usage

**All Operating Systems**

```
cd [project directory]
```
Then
```
node bot.js
```
To stop just type stop into the shell and enter.

## Versions

### v1.0.0
Basic bot, includes:
* Now playing (np)
* Pause
* Ping
* Play
* Queue
* Resume
* Skip
* Stop
* Volume (vol)

Bot will constantly updated until archived
